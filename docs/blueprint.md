# **App Name**: EnTrans

## Core Features:

- Transport Request Form: Users fill out a form specifying the type of transport needed (e.g., passenger, cargo), origin, destination, dates, and special requirements.
- Transport Offering: Companies (shippers) offer transport according to the characteristics the transport has.
- AI-powered transport matching Tool: The app intelligently matches user transport requests with available transport services. The tool will decide which properties are relevant in order to find the correct match.
- Real-time tracking: Once the transport is accepted, the app allow the transport owner and user to view in real-time where the transport is. This uses an external API.
- Notifications: The app provides push notifications and/or email alerts to users and transport providers about booking confirmations, status updates, and potential matches.
- Secure Payment System: Integrate a secure payment gateway to process payments between users and transport providers.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE), reminiscent of travel and open skies.
- Background color: Light gray (#F0F4F8), offering a clean, modern look.
- Accent color: Yellow-orange (#FFB74D), provides a contrast and evokes action/movement.
- Font pairing: 'Inter' (sans-serif) for body and 'Space Grotesk' (sans-serif) for headlines, to convey modernity and legibility.
- Use a consistent set of vector icons for transport types, locations, and features.
- Use a clean and intuitive layout with a clear visual hierarchy. Prioritize important information like transport details and statuses.
- Subtle transitions and animations for loading states and changes in transport status.
